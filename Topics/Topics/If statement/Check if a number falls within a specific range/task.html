Create a program that can determine whether a given number belongs to a specified range. Here are the rules: if the number is greater than or equal to 10 but less than or equal to 20, it belongs to the range. In this case, your program should print 'Inside the range'. If not, your program should print 'Outside the range'. Your program will be tested with a single number on each run.


Sample Input:
16

Sample Output:
Inside the range


Sample Input:
9

Sample Output:
Outside the range



Memory limit: 256 MB
Time limit: 15 seconds
